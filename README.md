# Sports Buddy

*Sports Buddy* is a web application that helps users find sports partners nearby, manage sports events, and provides an admin panel to manage categories, cities, and areas. The project uses *Firebase* for authentication and real-time database storage.

---

## Features

### User
- Register and login with email & password
- Add, update, and view sports events
- Logout functionality
- Responsive dashboard to manage events

### Admin
- Secure admin login
- Add/update categories, cities, and areas
- Manage app data dynamically

---

## Technologies Used
- *Frontend:* HTML, CSS, JavaScript
- *Backend / Database:* Firebase Authentication & Realtime Database
- *Development:* VS Code
